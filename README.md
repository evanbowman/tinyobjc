# tinyobjc
Tiny implementation of the GCC Objective C runtime, intended for microcontrollers. Requires GCC Objective C module version 8.

- [x] Loader for Classes and Metaclasses
- [x] Object root class
- [x] Method dispatch
- [x] Object allocation
- [ ] Method caching
- [ ] Unit tests
- [ ] Message forwarding
- [ ] Categories
- [ ] Protocols
- [ ] Some of the foundation classes
- [ ] Exception handling
